Planetary Gear System

Engineered a compact planetary gear system with optimized gear ratios

Tools Used

  • Fusion 360
  • Arduino
  • 3D Printing

Project Timeline

  • Duration: 2 months
  • Completed: August 2022

Key Features

  • Compact design
  • Optimized gear ratios
  • Integrated control system

Project Overview

This project involved designing and building a compact planetary gear system capable of handling high torque loads while maintaining precise control over output speed.

Using Fusion 360, I modeled the gear train and performed stress analysis to optimize tooth profiles and material selection. The system was prototyped using 3D printing for rapid iteration, with the final version featuring bronze alloy gears for durability.

An Arduino-based control system was integrated to monitor performance and adjust input speed dynamically. The completed system demonstrated a 90% efficiency rating in load tests.
