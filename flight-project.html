Flight Simulator Model

Modeled a dynamic flight simulator with realistic aerodynamics

Tools Used

  • MATLAB Programming and Simulation
  • Controls and Systems Integration
  • Dynamic Systems Modeling

Project Timeline

  • Duration: 4 months
  • Completed: December 2024

Key Features

  • Runge-Kutta Solver for Aircraft Motion
  • Quaternion-Based 3D Orientation
  • Arduino Joystick Control

Project Overview

I developed the flight simulator in MATLAB by implementing the Runge-Kutta method to solve the dynamic equations of motion for an aircraft, incorporating forces such as gravity, propeller thrust, and aerodynamic forces.

To accurately represent the airplane’s orientation in 3D space, I used quaternions for rotation calculations, which provided more efficient and stable handling of the simulation’s orientation. The system simulated real-time flight dynamics by applying these physical forces in the equations of motion.

Additionally, I integrated an Arduino joystick to control the airplane in the 3D animation, enhancing both the user interaction and the accuracy of the simulation.

This project refined my skills in coding, simulation, and systems integration, while deepening my understanding of flight dynamics, quaternions, and dynamic force modeling.
